Windows Registry Analysis: An Introduction

Evan Fogg
Posted on: February 6, 2020 at 5:24 PM

The Windows Registry, as confusing and complex as it is, can often be a forensic goldmine. More often than not the registry may leave behind a very revealing footprint of the user's actions. In this post I aim to explain how you can begin to use the registry as a powerful forensic tool to potentially aid in investigations.

The Structure of the Windows Registry

In order to understand the key paths I aim to showcase in this post, one must first understand the structure of the registry. The registry is composed of user specific "hives" that are stored as files on the disk. They are 5 main hives: HKEY_CLASSES_ROOT, HKEY_CURRENT_USER, HKEY_LOCAL_MACHINE, HKEY_USERS, and HKEY_CURRENT_CONFIG. These hives are logically viewed as one directory, and serve as the foundation of the registry.

The information in the Windows Registry is organized in a tree-structure with 3 main components: Keys (analogous to directory path), Values (analogous to filename), and Data (analogous to file content). This tree-structure begins with the root keys: HKEY_LOCAL_MACHINE (aka HKLM), HKEY_USERS (aka HKU), HKEY_CURRENT_USER (aka HKCU), HKEY_CURRENT_CONFIG (aka HKCC), and HKEY_CLASSES_ROOT.

Exploring the Registry

There are multiple methods to view or edit the registry. One may use the built-in regedit.exe tool to view registry contents live on their PC; however, this tool is not the best for registry forensics as it is primarily used to edit the registry. As such, it does not have the same search functionalities forensic registry explorers may have. The tool I recommend is Eric Zimmerman's Registry Explorer.

Important Keys to Check

In this section I aim to teach you the most common and important places to check. This varies heavily on the context of the investigation, but all of these key locations have the potential to be useful in one way or another in an investigation.

Computer Name

The computer name can be found at HKLM\System\CurrentControlSet\ Control\Computername

Current Version

The current version can be found at HKLM\Software\Microsoft\Windows NT\Currentversion

CPU Info (Vendor, Speed, Etc.)

CPU Info can be found at HKLM\System\CurrentControlSet\Enum\ACPI.

Time Zone Information

Time zone information can be found at HKLM\System\CurrentControlSet\Control\TimeZoneInformation.

List of Installed Applications

This list can be found at HKLM\Software\Microsoft\Windows\Currentversion\uninstall\{Application_name}

Autorun at System Bootup

Programs that are set to autorun on startup can either be in HKLM\Software\Microsoft\Windows\Currentversion\Run or HKLM\Software\Microsoft\Windows\Currentversion\RunOnce depending on whether or not they run multple times.

List of USB Devices

A list of USB devices used on the computer can be found at HKLM\System\CurrentControlSet\Enum\USB or HKLM\System\CurrentControlSet\Enum\USBSTOR if looking specifically for storage devices.
